coupons POST Interface

Method: POST

Relative URL: coupons/coupons-of-tab

Request Headers: Accept, Authorization

Response Headers: Content-Length, Content-Type.

Request Structure:

Field NameTypeDescription
couponsDetailsCouponDetailsInput

Response Structure:

Field NameTypeDescription
messageResponseMessagesuccess/error message

Example Request:

COUPONS 
Host: example.com
Authorization: Basic xxxxxxxxxxxxxxxxxxx
Content-Type : nnn
{
   "id":2147483647,
   "coupon":"?",
   "fromDate":"?",
   "toDate":"?"
}

Example Response:

{
   "id":2147483647,
   "status":"?",
   "message":"?"
}
