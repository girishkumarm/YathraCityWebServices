registerCar POST Interface

Developer Notes

Method: POST

Relative URL: cars/register-car

Request Headers: Accept, Authorization

Response Headers: Content-Length, Content-Type.

Request Structure:

Field NameTypeDescription
inputRegisterCarInputInput

Response Structure:

Field NameTypeDescription
messageResponseMessagesuccess/error message

Example Request:

REGISTERCAR 
Host: example.com
Authorization: Basic xxxxxxxxxxxxxxxxxxx
Content-Type : nnn
