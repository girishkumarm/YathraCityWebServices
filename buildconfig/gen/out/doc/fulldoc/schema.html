InBlox API Constructs: Persisted Entities

Contents


This section specifies the constructs upon which the API operates. The representations are made up of fields, each with a name and value, encoded using a JSON dictionary. The values may be numeric or string literals, lists, or dictionaries, each of which are represented as JSON elements.

Many of the model representations include a uri field whose value is the URI of the construct. This field type is present to support URI discovery in nested representations.
