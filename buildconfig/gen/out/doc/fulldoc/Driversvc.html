registerdriver POST Interface

Developer Notes

Method: POST

Relative URL: driver/register-driver

Request Headers: Accept, Authorization

Response Headers: Content-Length, Content-Type.

Request Structure:

Field NameTypeDescription
driverDetailsDriverDetailsdriver details to register a driver

Response Structure:

Field NameTypeDescription
responseResponseMessagesuccess/error message

Authorization: Basic xxxxxxxxxxxxxxxxxxx
Content-Type : nnn
